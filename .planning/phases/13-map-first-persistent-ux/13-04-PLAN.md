# Plan 13-04: Animations & Transitions

## Task

Add CSS transitions for stage changes and verify map zoom behavior.

## Implementation

1. Add overlay fade classes:
   - `.overlay-fade-enter`: opacity 0 → 1
   - `.overlay-fade-exit`: opacity 1 → 0
   - `.overlay-hidden`: visibility hidden, pointer-events none
2. Add marker transitions:
   - User pin: fade out when leaving setup
   - Venues: fade in with bounds
3. Test map.fitBounds() triggers on stage change

## CSS Structure

```css
.overlay-panel {
  transition: opacity 300ms ease-in-out, transform 300ms ease-in-out;
}

.overlay-panel.hidden {
  opacity: 0;
  pointer-events: none;
}
```
