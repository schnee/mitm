# Plan 13-02: Stage-Driven Overlay Panels

## Task

Create overlay panels that render on top of the persistent map for each stage with fade transitions.

## Implementation

1. Create `src/components/overlays/` directory
2. Create SetupOverlay:
   - Travel slider (50-100)
   - Tags selection
   - Submit button
   - Styled to layer on top of map
3. Create SpotsListOverlay:
   - Ranked results list (reuse from RankedResultsList)
   - Add to shortlist button
4. Create ShortlistOverlay:
   - Shortlisted venues
   - Confirm button
5. Create ConfirmOverlay:
   - Final venue display

## CSS Transitions

- Fade in/out with opacity + visibility
- 300ms transition duration
- Stage prop drives which overlay shows

## Panel Positioning

- Desktop: Right side overlay (300px width)
- Mobile: Bottom sheet overlay (full width, 40vh max)
